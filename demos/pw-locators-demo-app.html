Quick Reference Cheatsheet

All 7 locator strategies
getByRole('button', {name})Targets semantic ARIA roles — button, link, heading, checkbox, radio, table…
getByText('Welcome, John!')Matches visible text content; best for non-interactive elements like div, span, p.
getByLabel('Email Address')Links a form control to its <label> — the preferred way to target form inputs.
getByPlaceholder('Search...')Targets inputs via the placeholder attribute when no label exists.
getByAltText('Playwright logo')Finds images and area elements using their alt attribute text.
getByTitle('Total test runs')Matches elements with a matching title="" attribute (tooltip text).
getByTestId('search-button')The most resilient strategy — uses data-testid attributes added by developers.
